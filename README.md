# FileForge - Privacy-First File Processing Toolkit

FileForge is a privacy-first local file processing toolkit where users can compress, convert and manipulate files directly in their browser.

## Features

- **100% Client-Side Processing** - Your files never leave your device
- **No Accounts Required** - No signup, no login, no tracking
- **Image Tools** - Compress, convert, resize, and crop images
- **PDF Tools** - Compress, merge, split, and convert PDFs
- **PWA Support** - Install as a desktop app with offline support
- **Dark Mode** - Beautiful dark and light themes
- **Local Dashboard** - Track your file processing activity

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- browser-image-compression
- pdf-lib & pdfjs-dist
- JSZip & FileSaver
